# RNA Secagem de Grãos
Este repositório contém todos os arquivos e dados do meu projeto final para o curso de extensão "Redes Neurais Artificiais Aplicadas a Processos 
Químicos" na UNIFESP. Consistindo no treinamento de uma rede neural perceptron no Matlab utilizando a biblioteca Deep Learning Toolbox para interpolar a curva de cinética de secagem de grãos em temperaturas distintas.  

# Objetivo Principal
O objetivo principal desse projeto foi desenvolver uma rede neural com o papel de um sensor virtual. De forma que obtenha-se o tempo ideal de secagem por meio de parâmetros medidos antecedemente.
